READY FOR DEPLOYMENT
====================

Prerequisites:
  □ npm install
  □ Configure .env.local (see .env.example)
  □ Run migrations (see DEPLOYMENT.md)
  □ Seed database (scripts/seed-database.sql)

Deploy with:
  npm run build && vercel deploy --prod

Configuration needed:
  • AZURE_CLIENT_ID, AZURE_CLIENT_SECRET (Microsoft)
  • AZURE_TENANT_ID, AZURE_SHAREPOINT_* (SharePoint)
  • SENDGRID_API_KEY (Email)
  • TELEGRAM_BOT_TOKEN (Notifications)
  • MOBILE_API_KEY (Voice gateway)
